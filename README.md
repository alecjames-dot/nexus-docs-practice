# Nexus Docs

Product documentation for the Nexus protocol — a layer 1 blockchain built around a zkVM that generates validity proofs for every state transition.

## Quick Start

1. Check `PRODUCT_MAP.md` for the current status of any document before referencing it.
2. Read `GUIDELINES.md` to understand how this system works.
3. Read `CLAUDE.md` before using AI tooling on this repo.

## Products

- **Nexus Mainnet** — zkVM-based L1, validity proofs on every block
- **zkVM** — Zero-knowledge virtual machine with EVM compatibility
- **Prover Network** — Decentralized prover set with liveness slashing
- **USDX** — Native yield-bearing stablecoin sourced from on-chain revenue
