# UX Research

**Owner:** @alec | **Last updated:** 2026-03-02

This section contains UX research artifacts: interaction pattern analysis, user flow studies, and usability test results.

---

## What Goes Here

| Artifact Type | Description |
|--------------|-------------|
| Usability test reports | Findings from testing specific flows with real users |
| Flow analysis | Evaluation of existing or proposed user journeys |
| Interaction patterns | Reusable patterns and anti-patterns identified through research |
| Friction logs | Documented pain points in the current product experience |

---

## File Naming

```
USABILITY-TEST-[FEATURE]-[DATE].md      e.g., USABILITY-TEST-USDX-ONBOARDING-2026-03.md
FLOW-ANALYSIS-[FLOW]-[DATE].md          e.g., FLOW-ANALYSIS-PROVER-REGISTRATION-2026-03.md
FRICTION-LOG-[AREA]-[DATE].md
```
